tr

텍스트파일 내의 임의의 문자열을 원하는 문자로 바꾸거나 제거하는 명령, 원본의 파일내용이 바뀌지 않는 스트리밍 명령어이다.
보통 프로그램소스의 변수명이나 데이터 타입을 바꿀경우 유용하게 쓰인다. ( < 리다이렉션 명령어를 반드시 사용, 생략 불가능 )

tr [옵션] 문자1 문자2 < 파일명


*실제로 파일이 변경되지는 않는다

-d e   => 주어진 문자 "e" 를 삭제
-s ' ' => 문자가 중복된경우 한개만 남기고 삭제 (공백등을 제거할때)
